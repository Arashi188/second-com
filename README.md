# E-Commerce Product Catalog with WhatsApp Integration

A complete, production-ready e-commerce website built with Python Flask that allows customers to browse products and place orders via WhatsApp.

## Features

### Customer Side
- 🏠 Homepage with hero slideshow and featured products
- 📦 Product catalog with grid view and search
- 🔍 Product detail pages with stock status
- 🛒 Shopping cart system
- 📱 WhatsApp integration for order placement
- 📞 Contact page with business hours

### Admin Side
- 🔐 Secure admin login (hidden from public)
- 📊 Dashboard with statistics
- ➕ Add/Edit/Delete products
- 📸 Image upload with automatic resizing
- 📦 Stock management
- 📋 Order tracking system
- 📈 Low stock alerts

## Tech Stack

- **Backend**: Python, Flask
- **Database**: SQLite (development) / PostgreSQL (production)
- **ORM**: Flask-SQLAlchemy
- **Forms**: Flask-WTF
- **Authentication**: Flask-Login
- **Frontend**: HTML5, CSS3, JavaScript, Bootstrap 5
- **Image Processing**: Pillow
